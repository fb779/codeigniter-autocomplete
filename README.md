# Usage
1. Clone it somewhere in your project, like 'application/libraries/'
2. Add your models in fake_my_model.php
    <pre><code>
    class CI_My_Model {
        /**
         *
         * @var Org1_model
         */
        public $org1_model;
        /**
         *
         * @var Org2_model
         */
        public $org2_model;
    }
    </code></pre>
3. Refresh your project.
4. Enjoy yourself!
